## Site Structure Summary

The "Art by Gauri" website consists of the following pages:

### Core Pages
1. **Home** (`home.html`) - Hero, stats, featured artworks, classes preview, testimonials, CTA
2. **Portfolio** (`portfolio.html`) - Filterable artwork grid with categories
3. **Services** (`services.html`) - Portrait packages, commission process, classes overview
4. **Classes** (`classes.html`) - Class listings with online/offline filters
5. **Booking** (`booking.html`) - Portrait commission booking form
6. **About** (`about.html`) - Gauri's story, timeline, values
7. **Contact** (`contact.html`) - Contact form, WhatsApp, Instagram, location

### Dashboard Pages
8. **User Dashboard** (`dashboard.html`) - Orders, enrolled classes, profile
9. **Admin Dashboard** (`admin.html`) - Manage artworks, orders, classes, payments

### Design Features
- Dark, artistic theme (zinc-900 background, cream/gold accents)
- Tailwind CSS only (no custom CSS)
- Mobile-first responsive design
- Consistent header/footer across all pages
- Interactive elements with hover effects
- Ready for Laravel Blade integration

### Tech Stack
- HTML5 semantic markup
- Tailwind CSS via CDN
- Vanilla JavaScript for interactivity
- Google Fonts (Inter, Playfair Display)

All pages are modular and can be easily converted to Laravel Blade components.
